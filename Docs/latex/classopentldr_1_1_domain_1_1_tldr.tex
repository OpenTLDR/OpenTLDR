\doxysubsection{Detailed Description}
\begin{DoxyVerb}Tldr nodes represent an instance of the TLDR report for the date (in 'date' property of type
datetime.date). It is connected to the Request by the ResponseTo edge and a set of TldrEntries
by "Entries" edges.
\end{DoxyVerb}
